UX DESIGN

Você aprenderá a seguir o processo de design, compreender os fundamentos da pesquisa de UX, aplicar princípios de design centrado no usuário e criar um portfólio com projetos práticos.

Front-end

Este curso aborda as habilidades essenciais para criar interfaces de usuário atraentes e interativas. Você aprenderá a desenvolver sites e aplicativos web usando tecnologias como HTML, CSS e JavaScript.

Programação

Este curso é voltado para quem deseja se tornar um programador habilidoso. Você aprenderá lógica de programação, estruturas de dados e algoritmos, além de dominar uma ou mais linguagens de programação.

Data Science

Neste curso, você explorará técnicas para coletar, analisar e interpretar dados. Aprenderá sobre estatística, machine learning e visualização de dados.

Inteligência Artificial

Este curso abrange os fundamentos da IA, incluindo redes neurais, processamento de linguagem natural e visão computacional. Ideal para quem deseja criar sistemas inteligentes.

Desenvolvimento Mobile

Este curso foca no desenvolvimento de aplicativos para dispositivos móveis (iOS e Android). Você aprenderá a criar apps nativos ou híbridos.
